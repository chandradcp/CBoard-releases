# CBoard-releases

Public APK downloads for [CalBoard](https://github.com/chandradcp/CBoard) beta testers.

Source code stays in the private repo. This repo only hosts release APKs for in-app update (OTA).

## For testers

Install the latest APK from [Releases](https://github.com/chandradcp/CBoard-releases/releases). Debug builds can use **About → Check for updates** when a newer version is published.
